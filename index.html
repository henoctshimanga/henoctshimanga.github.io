Main Artifact

Original SmartCourse Finder

Developed in C++ as a console-based course management and advising system. It loaded course data from a file, stored courses in memory, displayed available courses, and supported course lookup and prerequisite viewing.

  • File-backed course data ingestion
  • In-memory storage and retrieval
  • Search and prerequisite display

Enhanced SmartCourse Finder

Rebuilt into a full-stack application using React frontend and Spring Boot backend. It now supports persistent theme state, graph-based prerequisite logic, MySQL/JPA persistence, and robust validation rules.

  • React UI with light/dark mode toggle
  • Spring Boot backend with RESTful services
  • MySQL database via Hibernate/JPA

Code Review & Enhancements

Frontend Enhancements

Added theme persistence and modern UI patterns with React. Verified the build with npm run build to ensure the user experience is stable and production-ready.

Backend Enhancements

Implemented Spring Data JPA repositories, validation logic, and MySQL persistence. I resolved a MySQL/Hibernate dialect compatibility startup issue, which reinforced the importance of accurate database configuration.

Algorithmic Enhancements

Moved beyond flat lookups by modeling course relationships as a graph. The system supports prerequisite validation, traversal logic, cycle rejection, and topological ordering for course dependency analysis.

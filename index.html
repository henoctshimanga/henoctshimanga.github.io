Artifact Overview

SmartCourse Finder is the core artifact that bridges original implementation and full-stack enhancements across frontend, backend, and database dimensions.

Original

Built as a C++ console application, SmartCourse Finder loaded course data from files, stored courses in memory, supported search, and displayed prerequisites.

  • Console-based interface
  • Flat in-memory course storage
  • File-driven data loading
  • Basic course lookup and prerequisite viewing

Enhanced

Rebuilt into a React + Spring Boot full-stack application with persistent theme state, graph-based prerequisite logic, and MySQL/JPA persistence.

  • Modern React frontend with light/dark theme
  • Spring Boot REST API and validation
  • MySQL persistence with Hibernate/JPA
  • Graph-based prerequisite validation and traversal
